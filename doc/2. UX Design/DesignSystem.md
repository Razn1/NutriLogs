Komponen Utama (Core Components)Tombol (Buttons)Primary: Background biru, teks putih, rounded 8px.Secondary: Border biru, teks biru, background transparan.Ghost: Tanpa border, teks abu-abu, hover effect subtle.Input FormState Normal: Border abu-abu terang, label di atas.State Focus: Border biru 2px, shadow halus.State Error: Border merah, pesan bantuan di bawah input.Kartu (Cards)Background putih, shadow tipis (0 4px 6px -1px rgb(0 0 0 / 0.1)), padding 24px.6.
